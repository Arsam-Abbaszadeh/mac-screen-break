# Screen break (for mac)
This is app for those who want to prevent themselves from getting lost in doom scrolling or doom web searching.
I an attempt to lower my base line level of dopmaine levels, I started specifcially trying to limit my time spent on dopaminergic activies (all of which are on my mac).
You can set a specific timer with this app after which all of your screens will be blocked not allowing further activity on your mac for a specific period of time.
